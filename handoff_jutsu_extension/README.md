# Handoff: Jutsu Downloader (расширение браузера)

## Overview
«Jutsu Downloader» — **расширение браузера** (Chrome, Manifest V3; совместимо с Edge/Brave/
Firefox), сфокусированная качалка аниме прямо внутри сайта **jut.su**. Пользователь сидит на
jut.su в своём браузере (уже залогинен под премиум-аккаунтом), расширение добавляет кнопки
«Скачать» и ведёт очередь. Само скачивание выполняет браузер через `chrome.downloads`.
Интерфейс — на русском. Основная тема — тёмная (сайт jut.su тёмный), есть светлая.

Это **не** отдельное приложение и не мини-браузер — вся работа идёт поверх реального
сайта в обычной вкладке. (Ср. с предыдущим бандлом `design_handoff_jutsu_downloader/` —
там десктопное приложение; здесь другой продукт с тем же дизайн-языком.)

Прототип покрывает **3 поверхности** + ключевые состояния + мини style-guide, с
переключением тёмной/светлой темы.

## About the Design Files
Файлы — **дизайн-референс в HTML** (прототип задуманного вида и поведения), а НЕ продакшн-код
для копирования. Задача — **воссоздать этот дизайн в среде расширения** (MV3): content
script, popup, options page, service worker. Обложки аниме нарисованы CSS-градиентами-
плейсхолдерами (в проде — реальные постеры с jut.su); jut.su под виджетом имитирован
статичной вёрсткой (в проде виджет монтируется поверх настоящей страницы).

- `Jutsu Extension.dc.html` — самодостаточный прототип (React-логика внутри).
- `support.js` — рантайм-шелл прототипа, для продакшена **не нужен**.

## Fidelity
**High-fidelity.** Финальные цвета, типографика, отступы, радиусы, тени, интерактив.
Прототип — pan/zoom-канвас: три поверхности в основных состояниях, затем ключевые состояния
и style-guide.

Размеры поверхностей:
- **Виджет (content script)**: панель ~344px, FAB 58px, открывается вверх от правого-нижнего угла.
- **Popup (тулбар)**: **360 × 480**.
- **Options**: отдельная вкладка, контент **720px** по центру.

## Архитектура MV3 (прод)

| Часть расширения | Роль |
| --- | --- |
| **content script** (на `*://jut.su/*`) | Инжектит FAB + панель поверх страницы. Определяет тип страницы (серия / аниме) по URL и DOM, распознаёт аниме·сезон·серию, парсит список серий. |
| **popup** (`action.default_popup`) | Очередь загрузок, быстрый выбор качества, недавнее. |
| **options** (`options_page`) | Настройки (качество, шаблон имени, папка, лимит, тема). |
| **service worker** (background) | Менеджер очереди; читает mp4-URL нужного качества со страницы серии по кукам сессии; ставит в `chrome.downloads`; шлёт прогресс в popup/widget; проверяет валидность сессии. |
| **storage** (`chrome.storage.sync/local`) | Настройки + история скачанного. |

Permissions (ориентир): `downloads`, `storage`, `scripting`, `activeTab`, host `*://jut.su/*`.
Скачивание — только `chrome.downloads.download({url, filename, conflictAction})`; имя строится
по шаблону из настроек.

## Design Tokens

Тема — через CSS-переменные на корневом контейнере (`data-theme="dark|light"`).

### Акцент и семантика (общие для обеих тем)
- Accent (primary): `#6C5CE7` · glow кнопок/FAB: `rgba(108,92,231,.4–.5)`
- Ссылки: `#8B7CF0` (dark) / `#5A4AD1` (light) · hover: `#A99CF7` / `#6C5CE7`
- Success / скачано: `#34D399`
- Queued / в очереди: `#F5A623`
- Error / отмена / бейдж-счётчик: `#FF5C5C`

### Поверхности (dark → light)
- Фон страницы/канваса: `#08080c` → `#EDEDF2`
- Поповер/виджет-панель/попап-меню: `#181820` → `#FFFFFF`
- Карточки, инпуты: `#15151C` / `#17171F` → `#F1F1F6`
- Чипы, дропдауны, мелкие кнопки: `#20202B` → `#ECECF2`
- Вторичные кнопки: `#1A1A24` / `#1E1E28` → `#ECECF2`
- Границы: `#22222E` → `#E2E2EA` · рамка поповера `#2A2A38` → `#DDDDE6`
- Трек прогресс-бара: `#26263480` (полупрозр.) → `#E4E4EC`
- Тумблер/чекбокс (off): `#3A3A4C` → `#C2C2CE`

### Текст (dark → light)
- Основной: `#EDEDF2` → `#1A1A22`

- Приглушённый: `#9A9AB0` / `#8A8A9E` / `#6B6B80` → `#6B6B7A` / `#7C7C8A` / `#9A9AA8`

### Тинты состояний (фон / граница / текст, dark → light)
- Success: `#1E3A28` / `#2E4A38` / `#34D399` → `#CDEED9` / `#BDE9CE` / `#1E9E63`
- Queued/info: `#2A2110` / `#2A2410` / `#F5A623` → `#FBF0D8` / … / `#F5A623`
- Error: `#2A1518` / `#4A2226` / `#FF5C5C` → `#FBE6E6` / `#F2C9C9`
- Accent-тинт (бейджи/тосты): `#241C42` / `#2E2751` / `#8B7CF0` → `#EBE8FB` / `#D8D2F5`

### Видеоплеер (всегда тёмный)
- `radial-gradient(120% 120% at 30% 20%, #2a1730, #0b0b10 70%)`; скраббер/оверлей `rgba(255,255,255,.2/.75)`

### Типографика — **Inter** (400/500/600/700/800), обязательна кириллица
- Display: 28px / 800 / −0.02em · Section title: 15px / 800 · Body: 13.5–14px / 600
- Caption: 12px / 500 · Eyebrow (uppercase, letter-spacing 0.1–0.14em): 11px / 700

### Радиусы
FAB 50% · панель/попап/карточки 11–16px · кнопки/инпуты 9–11px · чипы/бейджи 7–9px

### Тени (dark → light)
- Виджет-панель/попап: `0 26px 64px -14/-18px rgba(0,0,0,.75)` → `…rgba(0,0,0,.16)`
- Дропдауны: `0 18px 44px rgba(0,0,0,.6)` → `…rgba(30,30,60,.12)`
- Акцентные кнопки/FAB: `0 8–12px … rgba(108,92,231,.4–.5)`

### Spacing
Базовый ритм 4px. Паддинги панелей/карточек 11–18px, зазоры 8–14px.

Unicode-глифы (в проде — своя icon-библиотека): ⬇ ☰ ⚙ ▶ ✓ ✕ ❚❚ ▾ ⚠ ⓘ 🔒 🌐 📁

## Поверхность 1 — Виджет на странице jut.su (content script)

Плавающий круглый **FAB** (58px, accent, глиф ⬇) в правом-нижнем углу поверх сайта, с
красным бейджем-счётчиком активных загрузок. Клик → панель ~344px, открывается **вверх**.
Панель контекстно-зависимая по типу страницы:

### (A) Страница серии — `jut.su/<anime>/season-1/episode-5.html`
- Заголовок «Скачать эту серию» + подпись «Фейри Тейл · Сезон 1 · Серия 5».
- Дропдаун качества: **1080p Full HD / 720p / 480p / 360p**.
- Крупная accent-кнопка «⬇ Скачать серию».
- Вторичная ссылка «Скачать весь сезон (25) →».
- Внизу: компактный список активных загрузок (до 2 строк) — мини-прогресс, %, скорость,
  пауза/отмена; ссылка «Открыть очередь →» (ведёт в попап).

### (B) Страница аниме — `jut.su/<anime>/`
- Крупная кнопка «⬇ Скачать всё аниме» + счётчик-бейдж «353 серии».
- Ряд «Скачать сезон»: чипы «Сезон 1 … 4» (мультивыбор, accent при выборе).
- Дропдаун качества (общий для пачки).
- Тумблер «Пропустить уже скачанные» (**вкл по умолчанию**).
- Мелким: «В очереди: 8 · Готово: 40» + ссылка «Открыть очередь →».

Виджет читаем поверх тёмного сайта: свой фон `#181820`, рамка `#2A2A38`, мягкая тень —
заметный, но не кричащий.

## Поверхность 2 — Popup в тулбаре (360 × 480)
- **Шапка**: лого «Jutsu Downloader», справа ⚙ (→ options).
- **Качество по умолчанию** — сегмент-контрол 1080/720/480/360.
- **Активные загрузки**: заголовок + счётчик; сверху «Пауза всех» / «Отменить всё».
  Карточки-строки: мини-обложка 38px, «Аниме», «S1·E5 · 1080p», accent-прогресс-бар,
  «120 / 380 МБ · 22 МБ/с», кнопки ❚❚/✕.
- **Недавно скачано**: список с ✓ и кнопкой «Открыть папку».
- **Пустое состояние**: иконка + «Пока пусто — открой серию на jut.su и нажми ⬇».
- **«Вы не на jut.su»**: иконка + «Откройте jut.su, чтобы качать» + кнопка перехода.

## Поверхность 3 — Options (отдельная вкладка, контент 720px)
- **Качество по умолчанию** (сегмент).
- **Шаблон имени файла** — текст-инпут (`{anime}/Сезон {s}/{anime} S{s}E{e}.mp4`) с **живым
  превью** результата (плейсхолдеры `{anime}` `{s}` `{e}` подставляются).
- **Подпапка загрузок** — текст, по умолчанию «Jutsu» (внутри папки «Загрузки» браузера).
- **Одновременных загрузок** — степпер 1–5.
- **Пропускать уже скачанные** — тумблер.
- **Тема** — сегмент Тёмная / Светлая / Системная.

## Общие состояния
- **Тосты** (снизу по центру, автоскрытие ~3.4s): успех (зелёный тинт, ✓), инфо/завершение
  (accent-тинт, ⬇), ошибка/отмена (красный тинт, ✕).
- **Плашка «Сессия истекла / не залогинен»** — компактная (не модалка): ⚠ + текст + кнопка
  «Войдите на jut.su».
- **Качество недоступно → авто-понижение**: тост «Нет 1080p, качаю 720p» (queued-тинт, ⓘ).

## Interactions & Behavior
- **Скачать серию**: FAB → «Скачать эту серию» → выбор качества → «Скачать серию» → тост
  «Добавлено в очередь…». Прод: content script достаёт mp4-URL нужного качества по кукам
  сессии, service worker ставит в `chrome.downloads`.
- **Пакет**: на странице аниме — «Скачать всё» или выбор сезонов-чипов → качество → тост.
- **Очередь**: прогресс тикает; пауза/возобновление (❚❚/▶); отмена (✕, тост); по завершении —
  тост «Загрузка завершена», серия уходит из активных в «Недавно скачано».
- **Пауза всех / Отменить всё** — в шапке блока активных в попапе.
- **Тема**: мгновенно, `data-theme` на корне (все токены — CSS-переменные). Опция «Системная»
  следует `prefers-color-scheme`.
- **Анимации**: панель/дропдаун `jd-menu` .14–.16s (fade + translateY + scale); тосты
  `jd-toast` .25s; прогресс-бар `transition: width .5s linear`.

## State (прототип)
- `themeMode`: 'dark' | 'light' | 'system'
- `fabA`, `fabB`: открытость виджет-панели (по типу страницы)
- `qA` / `qB` / `popupQ` / `optQ`: выбранное качество (серия / пакет / попап / настройки)
- `seasons`: { [n]: bool } — выбранные сезоны на странице аниме
- `skip`, `optSkip`: «пропустить скачанные» (виджет / настройки)
- `optTpl`, `optSub`, `optConc`: шаблон имени, подпапка, лимит одновременных
- `downloads`: [{ id, animeId, title, sub, quality, done, total, speed, paused, active }]
- `recent`: [{ id, title, sub }]
- `toasts`: [{ id, icon, iconBg, iconColor, border, text }] (автоудаление по таймеру)

## Assets
- **Шрифт**: Inter (Google Fonts) — в проде локально-бандленый Inter, с кириллицей.
- **Обложки**: плейсхолдеры-градиенты → реальные постеры с jut.su.
- **Иконки**: Unicode-глифы → ваша icon-библиотека.
- Бренд jut.su принадлежит сайту; используется по факту отображения на странице.
